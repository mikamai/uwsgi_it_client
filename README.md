# UwsgiItClient

Ruby client for uwsgi.it API.

## Installation

Add this line to your application's Gemfile:

    gem 'uwsgi_it_client'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install uwsgi_it_client
